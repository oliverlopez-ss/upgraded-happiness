# Claude Instructions

## Research First

Before starting any implementation or configuration work, ALWAYS:

1. Research and read the official documentation for the tool, framework, or service involved
2. Verify that the approach is actually supported in the version being used
3. Confirm valid configuration keys/options exist before attempting to use them

Do NOT guess at configuration formats or try undocumented keys. Check the source docs first — this avoids hours of trial and error.
